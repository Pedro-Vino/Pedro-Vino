## 👨‍💻 Sobre mim

Sou um desenvolvedor focado em construir aplicações web robustas e escaláveis. Atualmente, estou aprofundando meus conhecimentos em **arquitetura de sistemas**, **autenticação segura** e **integração eficiente com bancos de dados**.

* 📚 **Na faculdade:** Consolidando a base lógica e algoritmos com a linguagem `C`.
* 🔥 **Foco atual:** Aperfeiçoando o desenvolvimento Back-end com o ecossistema `Node.js`.
* ⚡ **Diferencial:** Não apenas crio interfaces; foco na estruturação de sistemas reais, implementando regras de negócio complexas, autenticação de usuários e persistência de dados.

### 🎟️ SportAgora
> **Sistema web para gerenciamento de eventos esportivos e venda de ingressos.**
* **O que resolve:** Plataforma completa onde usuários podem se cadastrar, fazer login (seguro), visualizar eventos disponíveis e comprar ingressos (inteira ou meia).
* **Destaque Técnico:** Sistema robusto com autenticação (Bcrypt/Session), integração total com banco de dados MySQL para persistência e regras de negócio para categorias de ingresso, além de envio automatizado de e-mails.
* **Tecnologias:** <img src="https://img.shields.io/badge/Node.js-339933?style=flat-square&logo=node.js&logoColor=white" alt="Node.js" /> <img src="https://img.shields.io/badge/Express.js-000000?style=flat-square&logo=express&logoColor=white" alt="Express" /> <img src="https://img.shields.io/badge/MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white" alt="MySQL" /> <img src="https://img.shields.io/badge/EJS-B4CA65?style=flat-square&logo=ejs&logoColor=black" alt="EJS" />
